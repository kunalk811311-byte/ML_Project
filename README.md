# Customer Segmentation using Machine Learning

## Project Overview
This project performs customer segmentation on a telecom dataset using clustering techniques to identify distinct customer groups.

## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Customer segmentation using clustering
- Visualization of customer groups

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Dataset
Telecom customer dataset
